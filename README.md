# Todos API

## Overview

A Java Spring API project serving as the backend for a To Do application.

## Endpoints

- `GET /categories`
- `POST /categories`
- `PUT /categories/:id`
- `DELETE /categories/:id`

- `GET /todos`
- `GET /todos?category={}` query parameters
- `POST /todos`
- `PUT /todos/:id`
- `DELETE /todos/:id`
